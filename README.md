# Ejecurtar en desarrollo 

1. Clonar el repositorio 
2. Ejecutar

```
yarn install
``` 
 3. Tener Nest Cli intalado 
```
 npm i -g @nestjs/cli
 ```
 4. Levantar la base de datos 
 ```
docker-compose up -d
 ```

## Stack usado 
* Mongodb
* Nest
